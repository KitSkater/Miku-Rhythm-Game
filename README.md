# 🎵 Miku Rhythm Game

A simple DDR-style rhythm game featuring Hatsune Miku!

## 🕹️ How to Play

- Arrows scroll down the screen (← ↑ ↓ →)
- Hit them at the target zone to score!
- Miku cheers you on in the corner 💙

## 🎮 Controls

- Use the arrow keys to match notes
- Timing:
  - Perfect (±0.1s)
  - Good (±0.3s)
  - Miss
